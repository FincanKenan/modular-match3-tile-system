# 🧩 Modular Match-3 Tile System (Unity)

A modular Unity-based prototype demonstrating core match-3 / tile interaction mechanics.

---

Core interaction loop:
- Select tile  
- Move to tray  
- Match 3  
- Clear  

---

## 🧠 Core Mechanics

- Tile stacking system  
- Top-tile selection logic  
- Tray-based matching system  
- Match-3 detection and clearing  
- Modular and extensible architecture  

---

## 🏗️ Architecture Highlights

- ScriptableObject-driven data  
- Clean separation between logic and visuals  
- Scalable system for future mechanics  
- Designed for performance and flexibility  

---

## 🚀 Purpose

This project is a **system-focused prototype**, not a full game.

It demonstrates how a modern puzzle game (inspired by Royal Match / Zen Match) can be built with:
- clean architecture
- reusable systems
- scalable design

---

## 🛠 Tech Stack

- Unity (2D)
- C#
- ScriptableObjects
